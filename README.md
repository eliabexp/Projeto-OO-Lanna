# Trabalho prático - Orientação a Objetos

Este projeto foi desenvolvido por [Eliabe](https://github.com/eliabexp), [Gabriel](https://github.com/gabriellcardone-06) e [Igor](https://github.com/IgorDARAUJO) durante os dias 20/11/2025 até 03/12/2025. Foi utilizada a IDE [IntelliJ IDEA](https://www.jetbrains.com/idea) e o recurso Code With Me, que permitiu com que trabalhássemos no código ao mesmo tempo. Por esta razão, os commits só foram feitos por uma pessoa.

A aplicação visou seguir as orientações e exigências descritas [aqui](https://github.com/andrelanna/fga0158/tree/master/trabalhoPratico), e documentou mais informações sobre o projeto no relatório em PDF.
